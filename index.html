Селектор

Селектор (h1)- до чого буде пристасовано правило. До яких тегів, до яких класів, до яких id-ків.
Після селекторів йде тіло {} правила, перелік властивостей які ми змінюємо

Селектори за типами

Каскадність - (пристосується останнє правило для селектора)

Звернутись до класу

( . назва класу ///.text { })

Звернутись до id

(# назва id ///#number{ })

Поєднувати між собою декілько типів селекторів

(p.text{} spun#number{})

Кастомні атрибути тегів

Вага селекторів

  • tag name - 0001
  • class name - 0010
  • id name - 0100
  • inline style - 1000
  • inline style - 1000
  • important - 10 000

Псевдокласи, псевдоелементи

  • Псевдоклас h1:hover { }
  • Псевдоелемент h1::first letter { }
